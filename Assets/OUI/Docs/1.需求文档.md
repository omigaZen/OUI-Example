# OUI UI管理系统需求文档

目标：基于UGUI 简单易用的UI管理系统

- 依赖：UniTask

---

## 一、功能需求

### 1.1 Window管理

- 以Window为单位进行管理
- 约束：一个Window类只能有一个实例，重复打开已存在且已打开的Window直接return；资源正在加载中的Window也直接return
- 关闭Window不销毁实例，再次打开同一Window时复用已有实例（无需重新加载资源）
- 在Window类上加 `[WindowAttribute]` 标识Window对应的UI资源路径、Layer、HideBelow信息
- 资源加载通过 `IWindowAssetLoader` 接口解耦，由外部注入具体实现

### 1.2 层级排序

- 按WindowLayer分层，相同Layer的后打开的显示在前面
- 层级枚举：Bottom(0) → UI(1) → Top(2) → Tips(3) → System(4)

### 1.3 HideBelow遮挡

- 每次有窗口打开、关闭、销毁时，都需要重新计算遮挡：从栈顶向下遍历所有已打开的窗口，找到第一个 `HideBelow=true` 的窗口，该窗口及其上方的已打开窗口可见，该窗口下方的所有已打开窗口设置为不可见
- 已关闭的窗口不参与遮挡计算
- Window可通过回调感知自身被遮挡/恢复

## 二、公开接口

### 2.1 BaseWindow

```C#
public abstract class BaseWindow : MonoBehaviour
{
    /// <summary>
    /// 资源加载完成后调用一次，用于绑定UI组件引用。在Init()之前执行。
    /// </summary>
    public abstract void BindUI();

    /// <summary>
    /// BindUI()之后调用一次，用于初始化逻辑。
    /// </summary>
    public abstract void Init();

    /// <summary>
    /// 释放窗口资源时调用。
    /// </summary>
    public abstract void Release();

    /// <summary>
    /// 打开窗口时调用（首次打开和重新打开都会调用）。
    /// </summary>
    /// <param name="args">参数数组</param>
    public virtual void OnOpen(object[] args) { }

    /// <summary>
    /// 关闭窗口时调用（隐藏，不销毁）。
    /// </summary>
    public virtual void OnClose() { }

    /// <summary>
    /// 关闭自身的便捷方法（隐藏，不销毁）。
    /// </summary>
    public void Close();

    /// <summary>
    /// 窗口是否处于关闭状态。
    /// </summary>
    public bool IsClosed { get; }

    /// <summary>
    /// 是否被上层HideBelow窗口遮挡。
    /// </summary>
    public bool HideByAbove { get; }

    /// <summary>
    /// 遮挡状态变化时调用。
    /// 子类可重写以响应被遮挡/恢复事件（如暂停/恢复动画、音效等）。
    /// </summary>
    protected virtual void OnHideByAboveChanged(bool hidden) { }

    /// <summary>
    /// 按相对路径查找子Transform。找不到时抛出异常以便尽早暴露问题。
    /// </summary>
    protected Transform FindChild(string path);

    /// <summary>
    /// 按相对路径查找子节点上的组件。找不到时抛出异常以便尽早暴露问题。
    /// </summary>
    protected T FindChildComponent<T>(string path) where T : Component;
}
```


### 2.2 OUI（管理器）

```C#
public class OUI : MonoBehaviour
{
    public static OUI Instance { get; }

    /// <summary>
    /// 资源加载器，由外部注入。
    /// </summary>
    public IWindowAssetLoader AssetLoader;

    /// <summary>
    /// 异步打开窗口。首次打开会加载资源，已关闭的窗口会重新显示。
    /// 如果窗口已打开或资源正在加载中则直接return。
    /// </summary>
    /// <param name="args">传递给OnOpen的参数数组</param>
    public async UniTask ShowUIAsync<T>(params object[] args) where T : BaseWindow;

    /// <summary>
    /// 关闭指定类型的窗口（隐藏，不销毁，可再次ShowUIAsync打开）。
    /// 窗口不存在则抛出异常。
    /// </summary>
    public void CloseWindow<T>() where T : BaseWindow;

    /// <summary>
    /// 关闭指定类型的窗口（Type参数版本）。
    /// 窗口不存在则抛出异常。
    /// </summary>
    public void CloseWindow(Type type);

    /// <summary>
    /// 获取已存在的窗口实例（无论打开还是关闭状态），不存在则返回null。
    /// </summary>
    public T GetWindow<T>() where T : BaseWindow;

    /// <summary>
    /// 销毁指定类型的窗口（真正销毁GameObject和释放资源）。
    /// 窗口不存在则抛出异常。
    /// </summary>
    public void DestroyWindow<T>() where T : BaseWindow;

    /// <summary>
    /// 关闭所有窗口（隐藏，不销毁）。
    /// </summary>
    public void CloseAll();

    /// <summary>
    /// 销毁所有窗口（真正销毁）。
    /// </summary>
    public void DestroyAll();
}
```

### 2.3 资源加载接口

```C#
/// <summary>
/// 用于加载WindowAttribute标记的目标资源文件。
/// 由外部实现并注入到OUI中，解耦资源加载方案。
/// </summary>
public interface IWindowAssetLoader
{
    UniTask<GameObject> LoadAsync(string assetPath);
    void Unload(GameObject go);
}
```

### 2.4 WindowAttribute

```C#
public enum WindowLayer : int
{
    Bottom = 0,    // 最底层（背景等）
    UI = 1,        // 常规UI
    Top = 2,       // 置顶UI
    Tips = 3,      // 提示信息
    System = 4,    // 系统级（确认框等）
}

[AttributeUsage(AttributeTargets.Class)]
public class WindowAttribute : Attribute
{
    public readonly string AssetPath;
    public readonly WindowLayer Layer;
    public readonly bool HideBelow;

    public WindowAttribute(string assetPath, WindowLayer layer = WindowLayer.UI, bool hideBelow = true);
}
```

## 三、使用示例

### 3.1 定义窗口（无参数）

```C#
[WindowAttribute("UI/MainWindow", WindowLayer.UI, hideBelow: true)]
public class MainWindow : BaseWindow
{
    private Button m_btn_start;
    private TextMeshProUGUI m_tmp_title;

    public override void BindUI()
    {
        m_btn_start = FindChildComponent<Button>("m_btn_start");
        m_tmp_title = FindChildComponent<TextMeshProUGUI>("m_tmp_title");
        m_btn_start.onClick.AddListener(OnClickStart);
    }

    public override void Init() { }
    public override void Release() { }

    public override void OnOpen(object[] args)
    {
        m_tmp_title.text = "Welcome";
    }

    private void OnClickStart()
    {
        Close();
    }
}
```

### 3.2 定义窗口（带参数）

```C#
[WindowAttribute("UI/ShopWindow", WindowLayer.UI)]
public class ShopWindow : BaseWindow
{
    public struct Args
    {
        public int ShopId;
        public string Title;
    }

    private TextMeshProUGUI m_tmp_title;

    public override void BindUI()
    {
        m_tmp_title = FindChildComponent<TextMeshProUGUI>("m_tmp_title");
    }

    public override void Init() { }
    public override void Release() { }

    public override void OnOpen(object[] args)
    {
        if (args.Length > 0 && args[0] is Args shopArgs)
        {
            m_tmp_title.text = shopArgs.Title;
        }
    }
}
```

### 3.3 打开/关闭窗口

```C#
// 首次打开（加载资源 + 实例化 + 显示）
await OUI.Instance.ShowUIAsync<MainWindow>();

// 带参数打开
await OUI.Instance.ShowUIAsync<ShopWindow>(new ShopWindow.Args
{
    ShopId = 1,
    Title = "商店"
});

// 重复打开已打开的窗口，或资源加载中，直接return无任何效果

// 关闭（隐藏，不销毁，保留实例）
OUI.Instance.CloseWindow<MainWindow>();

// 再次打开（复用已有实例，无需重新加载）
await OUI.Instance.ShowUIAsync<MainWindow>();

// 获取已存在的窗口（无论打开还是关闭状态）
var existing = OUI.Instance.GetWindow<MainWindow>();

// 销毁（真正释放资源 + 销毁GameObject）
OUI.Instance.DestroyWindow<MainWindow>();

// 关闭所有（隐藏，无特定顺序）
OUI.Instance.CloseAll();

// 销毁所有（真正销毁，无特定顺序）
OUI.Instance.DestroyAll();
```

## 四、编辑器工具

### 代码生成

- Unity编辑器Hierarchy中节点右键菜单点击"生成OUI代码"，生成代码到剪贴板

#### 命名规则与类型映射

| 前缀 | 类型 | 示例 |
|------|------|------|
| `m_btn_` | Button | `m_btn_continue` |
| `m_img_` | Image | `m_img_bg` |
| `m_tmp_` | TextMeshProUGUI | `m_tmp_name` |
| `m_tf_` | Transform | `m_tf_hero` |
| `m_go_` | GameObject | `m_go_effect` |
| `m_item_` | GameObject（列表项模板，不递归扫描子节点） | `m_item_skill` |

#### 生成规则

- 绑定对象：递归查找子Transform中名称带有 `m_` 前缀的节点
- 路径：相对于当前选中节点的路径
- Button类型自动生成 `onClick.AddListener` 绑定代码，回调方法命名规则：`m_btn_xxx` → `OnClick_xxxBtn`
- **`m_item_` 特殊处理**：`m_item_` 前缀的节点视为列表项模板，其子Transform不再递归扫描 `m_` 标记

#### 生成代码示例

```C#

    m_btn_continue.onClick.AddListener(OnClick_continueBtn);
}

#endregion
```
