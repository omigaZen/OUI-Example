# OUI UI管理系统需求文档

目标：基于UGUI 简单易用的UI管理系统

- 依赖：UniTask

---

## 一、功能需求

### 1.1 Window管理

- 以Window为单位进行管理
- 约束：一个Window类只能有一个实例，重复打开已存在且已打开的Window直接return
- 关闭Window不销毁实例，再次打开同一Window时复用已有实例（无需重新加载资源）
- 在Window类上加 `[WindowAttribute]` 标识Window对应的UI资源路径、Layer、HideBelow信息
- 资源加载通过 `IWindowAssetLoader` 接口解耦，由外部注入具体实现

### 1.2 层级排序

- 按WindowLayer分层，相同Layer的后打开的显示在前面
- 层级枚举：Bottom(0) → UI(1) → Top(2) → Tips(3) → System(4)

### 1.3 HideBelow遮挡

- 标记了 `HideBelow=true` 的Window打开后，隐藏其下方所有Window以节省性能
- 该Window关闭后，下方Window恢复显示
- Window可通过回调感知自身被遮挡/恢复

## 二、公开接口

### 2.1 BaseWindow

```C#
public abstract class BaseWindow : MonoBehaviour
{
    /// <summary>
    /// window资源加载完成后，Init之前调用。用于绑定UI组件引用。
    /// </summary>
    public abstract void ScriptGenerator();

    /// <summary>
    /// ScriptGenerator()之后调用，用于初始化逻辑。
    /// </summary>
    public abstract void Init();

    /// <summary>
    /// 释放窗口资源时调用。
    /// </summary>
    public abstract void Release();

    /// <summary>
    /// 打开窗口时调用（首次打开和重新打开都会调用）。
    /// </summary>
    public virtual void OnOpen(params object[] args) { }

    /// <summary>
    /// 关闭窗口时调用（隐藏，不销毁）。
    /// </summary>
    public virtual void OnClose() { }

    /// <summary>
    /// 关闭自身的便捷方法（隐藏，不销毁）。
    /// </summary>
    public void Close();

    /// <summary>
    /// 窗口是否处于关闭状态。
    /// </summary>
    public bool IsClosed { get; }

    /// <summary>
    /// 是否被上层HideBelow窗口遮挡。
    /// </summary>
    public bool HideByAbove { get; }

    /// <summary>
    /// 遮挡状态变化时调用。
    /// 子类可重写以响应被遮挡/恢复事件（如暂停/恢复动画、音效等）。
    /// </summary>
    protected virtual void OnHideByAboveChanged(bool hidden) { }

    /// <summary>
    /// 按相对路径查找子Transform。
    /// </summary>
    protected Transform FindChild(string path);

    /// <summary>
    /// 按相对路径查找子节点上的组件。
    /// </summary>
    protected T FindChildComponent<T>(string path) where T : Component;
}
```

### 2.2 OUI（管理器）

```C#
public class OUI : MonoBehaviour
{
    public static OUI Instance { get; }

    /// <summary>
    /// 资源加载器，由外部注入。
    /// </summary>
    public IWindowAssetLoader AssetLoader;

    /// <summary>
    /// 异步打开窗口。首次打开会加载资源，已关闭的窗口会重新显示。
    /// 如果窗口已打开则直接返回已有实例。
    /// </summary>
    public async UniTask<T> ShowUIAsync<T>(params object[] args) where T : BaseWindow;

    /// <summary>
    /// 关闭指定类型的窗口（隐藏，不销毁，可再次ShowUIAsync打开）。
    /// </summary>
    public void CloseWindow<T>() where T : BaseWindow;

    /// <summary>
    /// 关闭指定类型的窗口（Type参数版本）。
    /// </summary>
    public void CloseWindow(Type type);

    /// <summary>
    /// 获取已存在的窗口实例（无论打开还是关闭状态），不存在则返回null。
    /// </summary>
    public T GetWindow<T>() where T : BaseWindow;

    /// <summary>
    /// 销毁指定类型的窗口（真正销毁GameObject和释放资源）。
    /// </summary>
    public void DestroyWindow<T>() where T : BaseWindow;

    /// <summary>
    /// 关闭所有窗口（隐藏，不销毁）。
    /// </summary>
    public void CloseAll();

    /// <summary>
    /// 销毁所有窗口（真正销毁）。
    /// </summary>
    public void DestroyAll();
}
```

### 2.3 资源加载接口

```C#
/// <summary>
/// 用于加载WindowAttribute标记的目标资源文件。
/// 由外部实现并注入到OUI中，解耦资源加载方案。
/// </summary>
public interface IWindowAssetLoader
{
    UniTask<GameObject> LoadWindowAsync(string location);
}
```

### 2.4 WindowAttribute

```C#
public enum WindowLayer : int
{
    Bottom = 0,    // 最底层（背景等）
    UI = 1,        // 常规UI
    Top = 2,       // 置顶UI
    Tips = 3,      // 提示信息
    System = 4,    // 系统级（确认框等）
}

[AttributeUsage(AttributeTargets.Class)]
public class WindowAttribute : Attribute
{
    public readonly WindowLayer WindowLayer;
    public readonly string Location;
    public readonly bool HideBelow;

    public WindowAttribute(WindowLayer windowLayer, string location = "", bool hideBelow = false);
}
```

## 三、使用示例

### 3.1 定义窗口

```C#
[Window(WindowLayer.UI, "UI/MainWindow", hideBelow: true)]
public class MainWindow : BaseWindow
{
    private Button m_btn_start;
    private TextMeshProUGUI m_tmp_title;

    public override void ScriptGenerator()
    {
        m_btn_start = FindChildComponent<Button>("m_btn_start");
        m_tmp_title = FindChildComponent<TextMeshProUGUI>("m_tmp_title");
        m_btn_start.onClick.AddListener(OnClickStart);
    }

    public override void Init() { }
    public override void Release() { }

    public override void OnOpen(params object[] args)
    {
        m_tmp_title.text = args.Length > 0 ? args[0] as string : "Welcome";
    }

    private void OnClickStart()
    {
        Close();
    }
}
```

### 3.2 打开/关闭窗口

```C#
// 首次打开（加载资源 + 实例化 + 显示）
var window = await OUI.Instance.ShowUIAsync<MainWindow>("Hello");

// 关闭（隐藏，不销毁，保留实例）
OUI.Instance.CloseWindow<MainWindow>();

// 再次打开（复用已有实例，无需重新加载）
var window2 = await OUI.Instance.ShowUIAsync<MainWindow>("World");
// window2 == window，同一个实例

// 获取已存在的窗口（无论打开还是关闭状态）
var existing = OUI.Instance.GetWindow<MainWindow>();

// 销毁（真正释放资源 + 销毁GameObject）
OUI.Instance.DestroyWindow<MainWindow>();

// 关闭所有（隐藏）
OUI.Instance.CloseAll();

// 销毁所有（真正销毁）
OUI.Instance.DestroyAll();
```

## 四、编辑器工具

### 代码生成

- Unity编辑器Hierarchy中节点右键菜单点击"生成OUI代码"，生成代码到剪贴板

#### 命名规则与类型映射

| 前缀 | 类型 | 示例 |
|------|------|------|
| `m_btn_` | Button | `m_btn_continue` |
| `m_img_` | Image | `m_img_bg` |
| `m_tmp_` | TextMeshProUGUI | `m_tmp_name` |
| `m_tf_` | Transform | `m_tf_hero` |
| `m_go_` | GameObject | `m_go_effect` |
| `m_item_` | GameObject | `m_item_skill` |

#### 生成规则

- 绑定对象：递归查找子Transform中名称带有 `m_` 前缀的节点
- 路径：相对于当前选中节点的路径
- Button类型自动生成 `onClick.AddListener` 绑定代码

#### 生成代码示例

